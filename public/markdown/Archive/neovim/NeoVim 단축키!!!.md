# NeoVim 단축키 정리

> NeoVim을 사용하면서 자주 사용하는 단축키들을 정리했습니다. 개발 효율성을 높이기 위해 꾸준히 연습하고 있습니다.

## 기본 단축키

### 버퍼 분할 및 터미널

- `<ALT + h>`: 버퍼를 가로로 분할하여 터미널을 엽니다.
- `<ALT + v>`: 버퍼를 세로로 분할하여 터미널을 엽니다.
- `<ALT + i>`: 플로팅 탭(가운데 창)에서 터미널을 엽니다.
- `<CTRL + n>`: 탭 바를 열고 닫을 수 있습니다.

### 버퍼 관리

- `<SHIFT + :vsplit>`: 첫 번째 버퍼를 명령으로 분할합니다. 오른쪽 버퍼를 선택하면, Tabufline의 두 번째 파일의 이름을 클릭하여 오른쪽 버퍼에 해당 파일이 열리고 나란히 작업할 수 있습니다.
- `<SHIFT + :options>`: Neovim의 모든 옵션을 볼 수 있습니다.
- `<SPACE + wK>`: 편집기에 포함된 모든 명령을 볼 수 있습니다. 나머지를 보려면 `d`를 누르면 됩니다.
- `<TAB>`: 드롭다운 메뉴가 열린 후에 TAB 키를 반복해서 이동할 수 있습니다.

**파일 열기 팁:**
`SHIFT + : e + SPACE` (edit) 명령 뒤에 경로를 입력하거나 명령 뒤에 TAB 키를 입력하여 버퍼에서 파일을 열 수 있습니다.

### 커서 이동

- `<j>`: 아래로 이동
- `<k>`: 위로 이동
- `<l>`: 오른쪽으로 이동
- `<h>`: 왼쪽으로 이동

## NvimTree - 파일 탐색기

### 기본 단축키

- `<CTRL + n>`: 탐색기를 엽니다. (NORMAL 모드에서만 사용 가능)
- `<CTRL + h>`: 편집기에서 탐색기 창으로 이동
- `<CTRL + l>`: 탐색기 창에서 편집기 창으로 이동

### 파일 탐색기 사용하기

프로젝트의 파일 트리 작업을 위해 NvimTree는 관리에 유용한 여러 바로 가기를 제공합니다:

- **R (refresh)**: 프로젝트에 포함된 파일을 다시 읽어 들이며, 변경도 가능합니다.
- **H (hide)**: 숨김 파일 및 폴더를 숨기거나 표시합니다. (점 `.`으로 시작하는 파일)
- **E (expand_all)**: root 폴더(워크스페이스)부터 전체 파일 트리를 펼칩니다.
- **W (collapse_all)**: root 폴더부터 열려 있는 모든 폴더를 닫습니다.
- **- (dir_up)**: 상위 폴더로 이동할 수 있습니다. 루트 폴더(워크스페이스)를 벗어나서 홈 디렉터리로 이동할 수도 있습니다.
- **s (system)**: 해당 파일 형식의 기본 시스템 응용 프로그램으로 파일을 엽니다.
- **f (find)**: 필터를 적용할 수 있는 대화식 파일 검색을 엽니다.
- **F**: 대화식 검색을 닫습니다.
- **Ctrl + k**: 파일의 크기, 생성 날짜 등과 같은 파일에 대한 정보를 표시합니다.
- **g + ?**: 빠른 참조를 위한 모든 미리 정의된 바로 가기를 포함한 도움말을 엽니다.
- **q**: 파일 탐색기를 닫습니다.
