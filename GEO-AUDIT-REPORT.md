## Critical Issues

1. **Cloudflare managed robots.txt가 AI 크롤러 차단** — 신규 존 기본값으로 GPTBot·ClaudeBot·Google-Extended·CCBot 등에 `Disallow: /` 주입. origin의 `Allow: /`와 같은 UA 그룹이 충돌해, 표준(RFC 9309) 파서는 허용하지만 보수적 파서는 차단으로 해석하는 **비결정 상태**. `Content-Signal: ai-train=no`도 origin 정책과 모순.
   → **사용자 액션 필요**: Cloudflare 대시보드 → toris.kr → AI Crawl Control(또는 Security→Bots)에서 "managed robots.txt / AI 봇 차단" OFF. (API 권한·보안정책상 자동 변경 불가)
2. **HTTP→HTTPS 미리다이렉트** — `http://toris.kr`이 평문 200 응답.
   → **사용자 액션 필요**: SSL/TLS → Edge Certificates → "Always Use HTTPS" ON. (HSTS 헤더는 ✅ `_headers`로 이미 적용)
3. ✅ ~~JSON-LD 날짜가 ISO 8601 아님(JS Date.toString)~~ — `toISO()` 정규화로 수정, 라이브 확인.
