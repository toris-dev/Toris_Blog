# 회원 탈퇴 정책 (MVP)

## 탈퇴 시 처리

1. 사용자가 속한 모든 그룹에서 **자동 탈퇴** (`leaveGroup`)
2. Firestore `users/{uid}` 문서는 삭제하지 않고 **익명화** (이름·이메일·프로필·FCM 토큰 제거, `deletedAt` 기록)
3. Firebase Authentication 계정 **삭제** (`deleteUser`)

## 제한

- **방장**이면서 그룹에 **다른 멤버가 있으면** 탈퇴 불가 — 방장 위임 또는 그룹 해산 후 재시도
- 최근 로그인이 오래된 경우 `auth/requires-recent-login` → 다시 로그인 후 탈퇴

## 보존 데이터 (MVP)

- 업로드한 사진·댓글·리액션은 그룹 콘텐츠로 **유지** (작성자 표시는 익명화된 프로필로 조회)
- R2 객체는 MVP에서 **자동 삭제하지 않음** (후속: Cloud Function 정리)

## 재가입

- 동일 이메일로 재가입 시 **새 UID**로 계정 생성 (이전 데이터와 자동 연결 없음)
