# 커플/그룹 실시간 공유 갤러리 서비스 기획서

---

# 1. 서비스 개요

## 서비스 한 줄 정의

커플, 가족, 친구, 소규모 그룹이 앱 내 카메라로 촬영한 순간을 즉시 공유하고 함께 추억을 쌓는 실시간 공유 갤러리 서비스

## 핵심 가치

- “찍는 순간 바로 함께 본다”
- 별도 업로드 없이 자동 공유
- 폐쇄형 프라이빗 공유 공간
- 관계 중심의 사진 아카이브

---

# 2. 문제 정의 (Pain Point)

현재 사용자들은:

- 카톡/디엠으로 사진을 따로 보내야 함
- 공유가 누락됨
- 사진 정리가 안 됨
- 앨범별 관리가 번거로움
- 커플/가족 사진이 여러 앱에 흩어짐
- 실시간 추억 공유 경험이 약함

특히 커플/가족은:

- “오늘 찍은 사진 서로 보내줘”
- “여행 사진 다 올려줘”
- “누가 찍은 사진인지 모르겠음”

같은 반복 행동이 발생함.

# A. 실시간 공유 카메라

## 핵심 기능

앱 내 카메라로 촬영 시:

- 자동 업로드
- 그룹원 즉시 확인 가능
- 실시간 동기화

> 기기 앨범·갤러리에서 기존 사진/동영상을 가져와 올리는 기능은 **의도적으로 제외**합니다. (촬영 순간 공유 UX)

## 사용자 경험

1. 앱 실행
2. 그룹 입장
3. 카메라 촬영
4. 즉시 공유 갤러리에 반영
