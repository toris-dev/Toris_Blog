# Inbox

정리 전 **캡처 전용**. 여기에만 쌓지 말고, 주 1~2회 아래로 분류한다.

## 캡처 규칙

- 한 줄이라도 OK (나중에 [[Projects]]·[[Planning]]·Daily로 이동)
- 링크·스크린샷·음성 메모 → 같은 날짜 Daily에도 한 줄 요약

## 분류 대상

| 종류 | 이동 위치 |
|------|-----------|
| 새 앱/서비스 아이디어 | [[Projects]] + `project` 템플릿 |
| 기능·요구사항 | 해당 프로젝트 + [[prd-checklist]] |
| UI·UX 메모 | [[Design]] 또는 프로젝트 디자인 노트 |
| 버그·기술 부채 | 프로젝트 + Daily |
| 배포·인프라 | [[Deployment]] |
