# 앱 이름 후보

- **미모Pick** (미모픽) — 쉽고 캐주얼, 선택/추천 플랫폼 느낌
- **뷰티Mate** (뷰티메이트) — 사용자와 병원을 이어주는 동행 이미지
- **닥터Pick** (닥터픽) — 신뢰 + 큐레이션, 병원 특화
- **예쁨Guide** (예쁨가이드) — 친절하고 초보자 친화적
- **클리닉Now** (클리닉나우) — 즉시성/예약/빠른 매칭
- **뷰티Flow** (뷰티플로우) — 상담→예약→관리 흐름
- **미모Log** (미모로그) — 기록/후기/변화관리
- **닥터Link** (닥터링크) — 연결/매칭 플랫폼
- **뷰티Bridge** (뷰티브릿지) — 병원-고객 연결
- **예쁨Pass** (예쁨패스) — 혜택/멤버십/정기관리 확장
