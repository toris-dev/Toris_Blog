# Journal (일일 기록)

하루마다 **목표 → 진행 → 회고**를 날짜별로 남기는 공간.

## 오늘 열기

- `Cmd/Ctrl + P` → **「오늘의 일일 노트 열기」** (또는 `Open today's daily note`)
- 사이드바 **Calendar** 플러그인에서 날짜 클릭

새 노트는 `journal/YYYY-MM-DD.md` 형식으로 자동 생성되며, [[daily]] 템플릿이 적용된다.

## 기록 루틴

| 시점 | 할 일 |
|------|--------|
| 시작 | 오늘 목표 1~3개 적기 |
| 작업 중 | **진행** 섹션에 한 줄씩 (프로젝트 `[[링크]]` 권장) |
| 마무리 | **완료** · **회고** · **내일** 채우기 |

자세한 개발 세션 흐름: [[dev-workflow]]

## 연결

- [[Inbox]] — 정리 전 캡처 → Daily로 이동
- [[Projects]] — 새 프로젝트 시작 시 Daily에 `시작: {이름}` 한 줄
- [[Home]] — 대시보드

## 파일 위치

```
journal/
  2026-06-03.md
  2026-06-04.md
  ...
```
