# PDF 글꼴

현장완료의 불변 A4 PDF 렌더러는 Google Fonts가 배포하는 나눔고딕 정적
TrueType을 사용합니다. `pdf-lib/fontkit`의 글리프 서브셋 생성과 호환되어
브라우저에서도 한글 PDF를 결정적으로 생성할 수 있습니다.

- `NanumGothic-Regular.ttf`
- `NanumGothic-Bold.ttf`

브라우저 UI는 Pretendard `v1.3.9`에서 생성한 압축 WOFF를 사용합니다.

- `Pretendard-Regular.woff`
- `Pretendard-SemiBold.woff`

원본:

<https://github.com/google/fonts/tree/main/ofl/nanumgothic>
