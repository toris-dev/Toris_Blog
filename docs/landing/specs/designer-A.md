# Designer A — 컨슈머·모바일 계열 4종 디자인 스펙

공통 전제: 모든 서비스 status '개발 중' → 1차 CTA는 사전 등록. 라이트 bg-slate-50 / 다크 dark:bg-[#050810] 페어, useReducedMotion() 시 모든 무한 루프·패럴랙스는 정적 최종 상태로 대체. 아이콘은 전부 인라인 SVG(stroke 1.5~2, currentColor).

## love-trip
Accent: #F43F5E → #FB7185, glow rgba(244,63,94,0.35) — 로맨틱 로즈. Web → 브라우저 목업.

### 히어로
- 헤드라인: "여행 계획, 3분이면 끝" — "3분"에만 rose 그라디언트 텍스트
- 서브카피: "출발지·예산·일정만 입력하면 교통편, 숙소, 데이트 코스까지 자동으로 완성됩니다."
- CTA: 1차 "사전 등록하고 먼저 쓰기" (rose 필, hover glow+scale 1.02) / 2차 "코스 미리 둘러보기" (ghost)
- 레이아웃: 좌 텍스트 45% / 우 브라우저 목업 55%. rose glow blob 2개 blur-3xl opacity-30, 스크롤 패럴랙스
- 목업 내부(플래너 폼): 주소창 "lovetrip.app/plan", 폼 — 출발 "서울 홍대입구", 도착 "부산 해운대", 날짜 "3.14 (토) – 3.15 (일)", 예산 "₩300,000" 슬라이더, 인원 "2명 · 커플" 칩, rose 버튼 "코스 만들기"

### 섹션 (6)
1. 문제 — "여행 준비, 탭이 몇 개나 열려 있나요?" 겹친 미니 탭 카드 4개(교통·숙소·맛집·정산)가 whileInView 시 rotate -6~+6deg stagger 0.1로 흩어짐. "교통 따로, 숙소 따로, 정산은 카톡으로."
2. 해결(폼→지도 전환) — "입력은 한 번, 계획은 완성" 좌 카피, 우 목업. 시그니처 A
3. 큐레이션 — "오늘은 어디 갈까?" 3열 카드(연남 로우커피 ★4.8 / 해운대 블루라인파크 / 부산현대미술관). stagger 0.08, hover lift y:-6
4. 1/N 정산 — "정산은 앱이 합니다" 좌 목업(영수증), 우 카피. 시그니처 B
5. 협업 — "둘이 같이 만드는 계획" 아바타 2개+커서 SVG 교대 이동(x 왕복 4s 루프). "상대가 담은 장소에 하트가 실시간으로"
6. 최종 CTA — "다음 여행은 love-trip과" rose glow, 이메일 인풋+버튼, whileInView scale 0.96→1

### 시그니처 인터랙션
- A. 폼→지도 전환: h-[220vh] 래퍼 + sticky 목업. useScroll progress 0→0.4 폼 opacity 1→0/x→-40, 0.3→0.7 지도(SVG 한반도+서울→부산 경로선) opacity 0→1, 경로선 pathLength [0.4,0.8]→[0,1], 0.7→1.0 핀 3개 scale 0→1 spring 순차 팝. reduced: 지도 완성 정적.
- B. 1/N 카운트업: whileInView(once) animate(0→127500, 1.2s) "₩127,500 / 인". 영수증 4줄(KTX ₩119,600 · 숙소 ₩89,000 · 저녁 ₩46,400) stagger 0.15 슬라이드 인, "정산 완료" rose 체크 뱃지 spring 팝.

### 목업 상세
- 지도: bg-rose-50 dark:bg-slate-800, 도로선 slate-300, 경로선 #F43F5E w3 dasharray, 핀 rose 물방울 SVG+흰 아이콘
- 일정 패널: "Day 1 — 10:00 KTX 출발 · 13:00 호텔 체크인 · 15:00 블루라인파크 · 19:00 해운대 야경"

### 컬러: glow blob·CTA·경로선/핀·체크 뱃지·eyebrow에 rose, 전체 10% 이내 절제.

## snapmate
Accent: #F59E0B → #FB923C — 앰버 즉흥. Mobile → 폰 목업.

### 히어로
- 헤드라인: "찍는 순간, 함께 본다" — "순간"에 앰버 그라디언트
- 서브카피: "앱 카메라로 찍으면 그룹 갤러리에 바로. 보내는 과정 없이, 찍는 게 곧 공유입니다."
- CTA: 1차 "출시 알림 받기" / 2차 "작동 방식 보기" (섹션 2 smooth scroll)
- 레이아웃: 좌 텍스트 / 우 폰 2대 겹침(뒤 폰 rotate 6deg, x:40, opacity 0.85). 마운트 시 y:40→0 stagger 0.15 spring
- 목업: 앞 폰 = 카메라 뷰파인더(앰버 그라디언트+컵 실루엣, "커플 갤러리 · 지수랑" 라벨, 앰버 셔터 링), 뒤 폰 = 갤러리("우리 둘 · 사진 128장", 3열 썸네일 9개, "방금" 앰버 뱃지)

### 섹션 (6)
1. 문제 — "찍고, 고르고, 보내고… 그러다 안 보내죠" 말풍선 3개("사진 보내줘~"/"아 맞다"/"3주 뒤: 아직도 안 보냄") stagger 0.3, 마지막 rotate -2deg
2. 해결(셔터→착지) — "셔터가 곧 전송 버튼" 중앙 대형 폰. 시그니처 A
3. 그룹 갤러리 — "우리끼리만 보는 앨범" 그룹 카드 리스트("커플 · 2명", "가족 · 5명 · 새 사진 12", "제주 여행 팟 · 4명")
4. 2대 동기화 — "내 폰에서 찍으면, 네 폰에 뜬다" 폰 2대 나란히. 시그니처 B
5. 신뢰 — "가볍고, 빠르고, 안전하게" 3열: 번개 "평균 1.2초 업로드", 자물쇠 "그룹 외 접근 차단", 구름 "원본 화질 그대로"
6. 최종 CTA — "다음 순간부터 함께 보세요" 앰버 glow, "출시 알림 받기" + "iOS · Android 동시 출시 예정"

### 시그니처
- A. 셔터→갤러리 착지: h-[250vh]+sticky. 0→0.25 뷰파인더, 0.25~0.35 셔터 scale 1→0.85→1+플래시 opacity 0→0.6→0, 0.35~0.7 사진 scale 1→0.28 그리드 첫 셀로 이동 borderRadius 0→8, 0.7~1 갤러리 크로스페이드+앰버 링+"지수님이 좋아합니다" 토스트.
- B. 2대 동기화: whileInView(once). 왼쪽 셔터 press → 점선 경로 pathLength 0→1(0.5s) 사진 미니카드 비행 → 오른쪽 갤러리 scale 0→1 spring(bounce 0.4) 착지 + "새 사진 1" 뱃지.

### 컬러: 셔터 링·뱃지·경로선·CTA·하이라이트 링에 앰버. 썸네일 그라디언트도 앰버 계열.

## loca
Accent: #06B6D4 → #22D3EE — 시안 도시. Mobile → 폰 목업.

### 히어로
- 헤드라인: "이 역, 지금 무슨 일?" — "지금"에 시안 그라디언트
- 서브카피: "지하철역 반경 500m, 같은 공간에 있는 사람들과 익명으로 나누는 실시간 대화."
- CTA: 1차 "사전 등록하기" / 2차 "내 역 미리 찾기"
- 레이아웃: 좌 텍스트 / 우 폰. 폰 뒤 500m 반경 펄스 상시. 배경 도시 그리드(1px slate, opacity 0.06)
- 목업: 헤더 "홍대입구역 · 참여 23명"(시안 dot pulse), 버블 3개 — "출구9" "2번 출구 버스킹 대박인데??" / "익명·라떼" "9번 출구 쪽 카페 자리 많아요" / "익명·막차" "혹시 지금 2호선 지연인가요" + 타이핑 인디케이터. 입력바+시안 전송 버튼

### 섹션 (6)
1. 문제 — "매일 스치는 사람들, 대화는 0" 노선 SVG 위 익명 dot 8개 교차 왕복 8s 루프. "같은 시간, 같은 역. 그런데 아무 연결도 없죠."
2. 해결(반경) — "딱 500m, 지금 여기의 이야기" 좌 목업(지도+펄스), 우 카피. 시그니처 B 주인공
3. 실시간 대화 — "대화는 이렇게 흘러갑니다" 중앙 대형 폰. 시그니처 A
4. 익명성 — "부담 없이, 익명으로" 3열: 마스크 "실명·프로필 불필요", 방패 "역을 벗어나면 자동 퇴장", 시계 "대화는 24시간 후 소멸"
5. 어디서나 — "당신의 역은 이미 열려 있어요" 역 칩 12개("강남 412명" "홍대입구 289명" "성수 175명" "잠실 143명"…) stagger 0.05 팝, hover 시안 border, 카운트업
6. 최종 CTA — "다음 출근길, 대화가 기다립니다" 시안 glow+미니 펄스

### 시그니처
- A. 채팅 버블 순차 등장: 진입(once, 0.4) 시 타이핑 인디케이터(0.8s, dot y:[0,-3,0]) → 버블 교체 opacity/y:12→0/scale 0.9→1 spring(400,25) → 5개 1.1s 간격. 내용: "지금 개찰구 앞 붕어빵 트럭 왔어요"→"실화냐 몇 번 출구요"→"3번이요 줄 벌써 김"→(내 버블) "가는 중ㅋㅋ"→"익명·붕어빵 님이 입장했습니다" 시스템 라인.
- B. 500m 펄스: 역 dot 기준 동심원 3개 scale:[0.3,1] opacity:[0.5,0] 2.4s repeat, delay i*0.8. 최대 반경 점선 원+"500m" 라벨. 유저 dot 6개 opacity [0.4,1,0.4] 3s 루프.

### 목업: 상대 버블 bg-slate-100 dark:bg-slate-800 rounded-bl-sm, 내 버블 시안 그라디언트 rounded-br-sm. 지도 블록 도로+역 심볼.
### 컬러: 라이브 인디케이터·내 버블·펄스·칩 hover·CTA에 시안. "지금 살아있는 것"에만.

## bubble-bible
Accent: #D97706 → #FBBF24 — 골드 성장. Mobile → 폰 목업.

### 히어로
- 헤드라인: "말씀이 습관이 되도록" — "습관"에 골드 그라디언트
- 서브카피: "읽고, 나누고, 함께 자라는 성경 앱. 오늘의 한 장이 포인트가 되고, 포인트가 꾸준함이 됩니다."
- CTA: 1차 "출시 알림 신청" / 2차 "성장 시스템 보기"
- 레이아웃: 좌 텍스트 / 우 폰 + 플로팅 뱃지 3개("7일 연속" 불꽃, "+200P" 골드 pill, "Lv.12 Disciple") y:[0,-8,0] 5~7s 부유
- 목업: 리더 화면 — "요한복음 3장", 본문 3절(16절 골드 하이라이트), 진행바 "오늘의 읽기 2/3장", FAB "+200P 받기"

### 섹션 (7)
1. 문제 — "작심삼일, 성경 읽기의 최대 난관" 체크리스트(1/1 ✓, 1/2 ✓, 1/3 —, 1/4 —) 체크 2개만 순차 표시. "3일째부터 멈춘 계획표, 익숙하시죠?"
2. 읽기 경험 — "읽기에만 집중되는 화면" 좌 목업(리더+폰트 설정 시트), 우 카피
3. 성장 시스템(핵심) — "읽을수록 자라는 나" 중앙 대형 폰(프로필). 시그니처 A+B

5. 커뮤니티 — "혼자 읽지 않도록" 피드 카드 2개: "QT나눔 · 은혜님 Lv.23 — '오늘 시편 23편에서…' 좋아요 41 · 댓글 12" / "찬양추천 · 다윗님 Lv.8 — 좋아요 18"
6. 그룹 랭킹 — "우리 교회, 함께 달리기" 랭킹 3행("1. 사랑교회 청년부 — 연속 21일" 크라운 / "2. 은혜셀 — 18일" / "3. 새벽기도팀 — 15일") 1위 골드 tint
7. 최종 CTA — "오늘 한 장부터 시작하세요" 골드 radial glow + "개역한글 · iOS · Android"
